# Try JGit

learn how to use JGit to manipulate git repository.

## Prerequisite
- Java 21
- Maven 

## 1. Create .env file first

```dotenv
GIT_USERNAME=your_git_user_name
GIT_PASSWORD=your_git_password
```
## 2. Don't forget to modify uri
in JgitStarterApplicationTests.java

## 3. Run the test
```shell
mvn test
```
